---

![DNS картинка](/dns.png)

**DNS** _(Domain Name System)_ — в переводе с английского значит система доменных имен. Это специальная компьютерная распределённая система для получения информации о доменах. DNS-сервер (Name Server, NS) — это сервер, который преобразует домены в понятные для компьютеров IP-адреса и наоборот.
